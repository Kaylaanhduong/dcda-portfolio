Semester Reflection

What I learned: In Lab 1, I learned how to set up a project using GitHub for the first time and how to work with HTML files in VS Code. I also learned the differences between pushing and pulling changes, how to comment my code, and how GitHub tracks edits, making it easier to trace mistakes and understand my workflow

Challenges I overcame: Setting everything up without a video walkthrough or in-class guidance was confusing at first, and I struggled to understand how to open my HTML file correctly in VS Code. I felt frustrated and wanted to give up several times, but after realizing I needed to open the project directly from GitHub, things began to click. Working through these challenges helped me build confidence with unfamiliar technical tools.
